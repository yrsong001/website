# Assets

- **photo.jpg** – Your profile photo for the site. From the `resource/` folder, only this image is used on the published site. To add it: open `resource/Photo_Yiran.pdf` in Preview (or another viewer), export the first page as JPEG, and save as `assets/photo.jpg`. A square or cropped-to-square image works best for the circular avatar. If `photo.jpg` is missing, the site shows a gray circle.
- Social icons (Google Scholar, GitHub, LinkedIn, X) are in `index.html`. The Google Scholar link uses a placeholder—replace `YOUR_ID` in the URL with your Google Scholar user ID.
